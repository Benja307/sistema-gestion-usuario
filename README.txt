Comandos a ejecutar:

El comando "make" sirve para compilar, donde busca el archivo makefile, encargado de 
ejecutar las ordenes contenidas en el mismo:
$ make

Luego, el siguiente comando sirve para ejecutar el archivo Principal:
$ ./Principal
